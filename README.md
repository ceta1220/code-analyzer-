# Code Analyzer              

비밀번호의 안전성을 분석하고, 더 강력한 비밀번호를 추천하며, 랜덤 비밀번호를 생성하는 C 언어 기반 프로그램

기능
## 1. 비밀번호 안전성 분석

입력한 비밀번호의 길이와 문자 종류를 바탕으로 총 가짓수의 로그를 계산하여 안전성을 평가합니다.

분석 요소: 비밀번호 길이, 대문자 포함 여부, 소문자 포함 여부, 숫자 포함 여부, 특수문자 포함 여부

## 2. 안전한 비밀번호 추천

입력한 비밀번호를 분석하여 부족한 요소를 자동으로 보완합니다.

추가되는 요소:
대문자
소문자
숫자
특수문자
최소 길이 충족용 랜덤 문자

## 3. 랜덤 비밀번호 생성

사용자가 원하는 길이의 비밀번호를 생성합니다.
대문자 소문자 숫자 특수문자를 무작위로 조합하여 생성합니다.

## 4. 비밀번호 가짓수 계산

비밀번호 길이와 사용된 문자 종류를 기반으로 가능한 조합 수를 계산합니다.


# 실행 방법
코드를 실행하고 원하는 메뉴(기능)의 번호를 입력하여 실행
